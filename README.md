## Description
 This is an Express application that uses a new Mongo database, with a Mongoose schema to aid users in tracking their daily workouts. User are able to view, create and track daily workouts. Users are able to log multiple exercises in a workout. Each exercise is able to have the name, type, weight, sets, reps and duration entered and saved. If the exercise is a cardio exercise, the user will be able to track the total distance traveled.

## Installation
To install this on your local machine, please follow the steps below. 

1. Download this repository `workout-tracker`.

2. Install the basic npm packages by running the following command in your bash/terminal:

````
npm i -y
````

3. Install required dependencies by invoking the commands below. These include Express and Mongoose NPM Packages.

````
npm i express
npm i mongoose
````
Congratulations! You have successfully installed the workout tracker application in your local computer.
